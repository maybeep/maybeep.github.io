HTML Reference Guide (in HTML!)

Basic Tags

  • <html> </html> - Creates an HTML document (ancestor of all HTML elements)
  • <head> </head> - Section for the title & other info not displayed
  • <body> </body> - Section for the visible portion of the document

Text Tags

  • <pre> </pre> - Creates preformatted text
  • <h1> </h1> - Creates headlines from h1 (largest) to h6 (smallest)
  • <strong> </strong> - Creates bold text
  • <em> </em> - Creates italicized text
  • <code> </code> - Used to define source code
